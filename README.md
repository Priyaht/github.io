# IJC437 – Passenger Assist Accessibility Analysis

## Files
- `sample.R` – main analysis script

## How to run
1. Open `sample.R` in RStudio.
2. Install required packages:
   install.packages(c("tidyverse","readODS","janitor","ggplot2"))
3. Run the script.

## Data
This project uses ORR Passenger Assist tables.
If datasets are not included, download them from the ORR website and place them in the same folder as the script.
